# 💰 Expense Tracker App

An Android application designed to help users track their daily income and expenses efficiently. The app provides a clean interface to add, view, and manage financial transactions with real-time updates.

---

## Features

- ➕ Add income and expense transactions
- 📋 View all transactions in a structured list
- 🗂️ Categorize transactions (Food, Travel, Bills, etc.)
- 💵 Automatic calculation of formatted currency values
- 🎨 Clean and modern Material UI design
- 📊 Real-time updates using LiveData
- 🗄️ Local storage using Room Database

---

## Tech Stack

- **Language:** Kotlin
- **UI:** XML + Material Design Components
- **Architecture:** MVVM (basic structure)
- **Database:** Room Persistence Library
- **Async Handling:** Kotlin Coroutines
- **RecyclerView:** For transaction listing
- **KSP:** For Room annotation processing

---
